# 第二轮纸面原型：两代接力局面

状态：Prototype（结构已确认；概率与数值仅为本轮测试参数）

## 本轮迭代表

| 项目 | 内容 |
| --- | --- |
| 体验目标 | 玩家在具体人物有限寿元下分配唯一延寿机会与共同资源，承受第一次大考，并在第二代清楚看见上一代选择造成的优势、缺口与遗憾。玩家应该感到：“我救不了所有人，也做不完所有事；我现在保住或放弃的具体人物与成果，会真实决定下一代面对什么。” |
| 当前问题 | 具名人物竞争是否形成真实取舍；大考押注是否有意义；死亡、成果与未竟事业是否改变第二代开局；旧压力重临时是否出现可归因的碾压、改善或遗留问题。 |
| 备选方案 | 延寿岳徊、沈砺、归元真人或保留；庚金髓投入护山阵、苏砚培养或归元突破；大考选择守山门、保基业或反攻，并选择攻、防或均衡姿态。 |
| 采用方案 | 用十八个月、六个决策节点的一份两代接力局面完成阶段 2 收口。 |
| 最大风险 | 出现跨人物、跨局面固定答案；大考档位只有名义差别；第二代近似重置；所有路线最终仍可由一个总效率分排序。 |
| 原型与测试 | 闭环初稿已完成；内部全流程推演本次尚未执行，仍是阶段 2 后续必做步骤，按后续安排进行。届时主 Codex 完整内部推演一次，只排除矛盾和歧义；随后由用户真人试玩一次。观察分配理由、风险感受、人物在意程度、跨代因果可读性与是否愿意继续经营。 |
| 结果与新问题 | 待真人试玩后填写：保留什么、删除什么、阶段 2 通过或失败、阶段 3 是否解锁。 |

## 局面边界

- 总时长十八个月，纸面回合仍按月聚合。
- 只在第 1、3、6、9、15、18 月停下来作关键决定或结算；未修改安排自动延续。
- 第 1—6 月：上一代准备；第 6 月末第一次赤潮兽灾。
- 第 7—9 月：恢复伤势、寿元到期、成果与未完成事业交接；第 9 月生成第二代开局卡。
- 第 10—15 月：第二代经营；第 16—18 月旧压力再次出现，按第一次大考结果分支（见下节与决策台账 D-53）。
- 山门陷落则立即结束本局。第 18 月完成第二次大考后，本纸面原型结束。

### 月度与节点结算顺序

每月及关键节点统一按下列顺序结算（见决策台账 D-57）：

1. **寿命检测**：人物寿元归零前先公开提示，玩家可以使用延寿丹；这类提示可在固定节点之间出现，但不新增常规决策节点。所谓“第 N 月末寿尽”的人物，在第 N 月寿命检测时尚有最后一个月，可以完成第 N 月行动，并在符合其他参战条件时参加当月大考；第 N 月末寿元扣减为 0，进入第 N+1 月寿命检测时若仍未延寿才死亡，不能执行第 N+1 月行动或参战。
2. **行动检测**：结算当月行动、项目进度和已经完成的成果。正在整理功法、炼器或培养的人物在行动检测完成后可以参加当月大考；仍在闭关突破、重伤或仍在秘境无法赶回者不能参战。
3. **大考检测**：读取行动检测后的实际人物、成果与防务状态，结算押注、战略姿态、结果与参战者伤势。

受伤人物默认自动转为休养，恢复后原安排自动继续。玩家可以强制带伤工作，但会加重病情并降低效率；具体加重概率和效率下降幅度尚未决定。

| --- | --- | --- |

## 共享资源

### 唯一延寿丹

- 开局一枚，服用者增加十二个月寿元。
- 第 3 月末前可以给岳徊；若不给，岳徊寿尽，丹药保留。
- 第 5 月末前可以给沈砺；若仍保留，第 6 月末前可给归元真人，亦可继续保留。
- 延寿只增加时间，不自动完成资格、熟练度、功法整理、突破或阵器。

### 庚金髓

- 开局三份。
- 岳徊在世且安排采集时，每月历练最多结算两份；资源量不随队伍人数增加，合格队员只通过战力降低公开的受伤风险。
- 岳徊第 3 月末寿尽；获得延寿后，第 4—6 月可以继续进入沉锋谷采集，仍按每月最多 2 份结算（见决策台账 D-54）。
- 若第 1—6 月连续采集，第一次大考前开局 3 份 + 最多新增 12 份 = 最多 15 份。此前“最多九份 / 局面内最多增加六份”的表述废止。
- 资源虽增加，但到达月份与连续三个月工期仍限制大考前能完成的项目组合，不能把全部资源视为第 1 月可用。
- 护山阵器：开工时一次投入六份 + 沈砺连续三个月。
- 苏砚硬资格培养：三份 + 连续三个月（见下节风险参数）。
- 归元真人突破辅助：三份，使突破准备由尚可提升到充分。
- 三项目共需十二份。延寿后采集可把第一次大考前可用量抬到最多 15 份，但工期与到达月份仍约束能同时完成的组合；岳徊若转去整理引路图，第一次大考前可用庚金髓进一步减少。

### 苏砚硬资格培养（第二轮纸面参数）

- 工期连续三个月，开始时投入三份庚金髓；失败后第一次大考前没有时间重试。
- 公开结果概率（第二轮临时参数，非最终平衡；见决策台账 D-55）：70% 成功 / 27% 受伤且未取得资格 / 3% 重伤且未取得资格 / 0% 死亡。
- 包含风险但不会死亡；成功才取得硬资格，受伤/重伤均未取得资格。
- 与 D-42 突破尚可档不是同一系统；本条只服务苏砚资格培养。

### 当前境界突破成功延寿（第二轮纸面参数）

- 当前纸面境界突破成功暂定增加 24 个月寿元（见决策台账 D-56）。
- 只适用于第二轮当前境界纸面参数，不扩成所有境界统一规则或最终平衡。

## 第一次大考：赤潮兽灾

### 玩家押注

| 目标 | 要保护的内容 | 可得收益 |
| --- | --- | --- |
| 守山门 | 只确保宗门核心，允许放弃外部产业 | 无正向资源兑现；对应 C 档净损失语义 |
| 保基业 | 山门与全部外部产业 | 第一层通用战利品 + 第二层专业战利品；通用层足以明显加快第二代并支撑多个重要项目（具体清单与数量待定） |
| 反攻妖巢 | 保住全部目标并追击赤鳞妖王 | 在 B 的两层之上再取得第三层稀有战略成果；第二代前半段普通资源约束大幅解除，并开启持续发展机会；人物风险更高（具体清单与数量待定） |

### 正向奖励三层结构（关系已确认；名称与数值待定）

大考正向奖励不压成单一通用货币，分成三层（见决策台账 D-52）：

1. **通用战利品**：任何经营路线都能立即使用；大考成功后宗门必须明显富起来。
2. **专业战利品**：需要炼器者、炼丹者或其他专业人物才能充分转化；丰收之后专业人物的存亡仍重要。
3. **稀有战略成果**：仅 A“反攻告捷”取得，例如妖王内丹、资源产地或长期通路一类结果，用于开启新的长期发展能力。

边界：B 含第一、二层，不含第三层；A 在 B 之上再含第三层；C 无正向资源兑现；D 终局。资源不能买回死亡人物的专业能力、个人经验或继任资格。具体资源名称、数量、比例、项目数、产量与加工公式尚未决定，本轮不自行扩展。

### 四档结果

| 档位 | 结果 |
| --- | --- |
| A 反攻告捷 | 山门、外部产业完整；获得第一、二层奖励，并额外取得第三层稀有战略成果；第二代前半段普通资源约束大幅解除，同时获得稀有资源与持续发展机会（具体内容待定） |

| C 山门独存 | 山门保住，外部产业损坏或丢失；无正向资源兑现，是净损失结果，人员伤亡与第二代恢复负担增加 |
| D 山门陷落 | 本局立即失败并结束；逃生者仅进入流亡后传，不继续游玩 |

### 防务画像

- 防务完备：护山阵完成，并有健康的筑基圆满或突破后的归元真人。
- 一处缺口：有阵但归元受伤/缺席且仍有具名支援；或无阵但归元健康/突破成功。
- 两处缺口：有阵但归元重伤/死亡且支援不足；或无阵且归元受伤。
- 濒临崩溃：无阵，并且归元重伤、死亡或无法参战。

### 战略姿态修正

- 攻守兼备：直接使用上表。
- 全力进攻：守山门时最多把 20 个百分点从 D 移到 C；保基业时最多把 10 点从 C 移到 B，若基础 D 大于 0 且 C 至少还剩 5 点，再把 5 点从 C 移到 D；反攻时若基础 A 大于 0，最多把 15 点移到 A，明确先从 C 扣除、不足部分再从 B 扣除，随后若基础 D 大于 0，再把 5 点从剩余 C 移到 D。基础 A 为 0 时不能凭姿态解锁 A；基础 D 为 0 时不新增小概率硬失败。
- 全力防守：不能选择反攻妖巢。守山门时最多把 10 点从 D 移到 C；保基业时先最多把 10 点从 D 移到 C，再最多把 15 点从 B 移到 C，表示以外部产业换人物与山门安全。

## 第 9 月交接卡

交接不把人物死亡转换为宗门底蕴，也不计算统一总分。第 7—9 月应逐项记录：人物是否仍在、已经完成并外化的永久成果、可以被后来者学习但仍需条件的知识、尚未完成且仍可接续的事业、因人物死亡或准备不足而永久失去的能力与机会。

下文固定五名人物与护山阵、归元功法、沉锋谷通路、庚金髓、外部产业在 A/B/C 三种存续结果下的交接关系；D 已直接终局，不进入交接。

### 交接规则初稿（待用户确认）

第 9 月不做“继承结算”，而是对仍存在的事实重新盘点。每项内容只能落入下列一栏，避免同一价值重复计算：

| 栏位 | 含义 | 第二代如何使用 |
| --- | --- | --- |
| 在世人物 | 人还活着，其境界、战力、熟练度、资格与寿元仍属于本人 | 可以继续安排；死亡后不会自动转为宗门资产 |
| 永久成果 | 已经建成、炼成或储存并脱离个人存在的实物 | 只要山门和该实物未被摧毁即可继续使用 |
| 可携知识 | 已经写成完整功法、图纸或引路图 | 后人仍须满足适配、境界与学习条件，不等于自动掌握 |
| 可接续未竟事业 | 已留下足够记录，且当前存在合格承接者 | 从已记录进度继续；缺少其中任一条件则不能接续 |
| 永久失去 | 只存在于死者身上的能力、经验或没有留下可用记录的未完成工作 | 第二轮十八个月内不再通过同名替代者补回 |

### 五名人物的最小交接关系

| 人物 | 人物存活时 | 人物死亡时 |
| --- | --- | --- |
| 归元真人 | 其战力、境界、寿元和亲自掌握的功法仍是个人能力；可以继续守山、授业或追求仙途 | 完整整理的《归元剑典》进入可携知识，林砚仍须学习；突破失败只留下残卷，其补齐工期取决于归元生前的实际整理进度与残缺比例，补齐期间不能形成完整功法战力；未整理且未留下残卷的个人理解永久失去 |
| 林砚 | 保留自己已经学会的内容与个人成长，不因归元存亡自动升级 | 若在第一次大考死亡，他已经学会但未外化的部分随之消失；功法文本仍可留给未来其他适配者，但本轮不自动生成新人 |
| 岳徊 | 保留沉锋谷资格、熟练度和稳定引路能力，可继续带领已经合格的苏砚 | 完整引路图进入可携知识；苏砚若已经合格并有带队经历，保留自己的熟练度；只有图而无合格继任者时通路暂时停摆；既无图也无成熟经历时，岳徊的路线经验永久失去，本轮不能稳定进入沉锋谷 |
| 苏砚 | 已完成的金灵根适配、境界、耐力、战力与秘境熟练度都属于本人 | 资格培养和熟练度随本人死亡消失；引路图仍存在，但本轮不自动出现第二名稀有适配者 |
| 沈砺 | 保留炼器熟练度、个人战力、未来高阶生产与继续修补阵器的能力；C 档受损护山阵可由其修复 | 已完成的护山阵作为永久成果留下，但若第一次大考为 C 则进入受损状态；修复必须有专门的合格炼器者，本轮默认沈砺存活时具备资格。沈砺死亡且没有培养出合格承接者时，受损阵器仍保留，但本轮无法恢复完整效力。未完成阵器只有在生前写成可执行图纸且存在合格炼器承接者时才能继续；本轮五人中没有同等承接者，因此未完成且未建成的部分永久失去。死亡本身不产生传承 |

### 第 9 月生成的第二代开局卡

### 功法学习与残卷补齐（第二轮纸面参数）

- 完整功法具有自己的基础学习工期；原作者亲授使学习速度变为两倍、所需工期减半。
- 本轮《归元剑典》自行研习需要十二个月；由归元真人亲授需要六个月。
- 残卷补齐工期按归元真人生前的实际完成进度与残缺比例确定，不固定为六个月。补齐后才成为完整功法，后续学习与完整《归元剑典》使用同一规则。
- 因此，无归元亲授时，林砚在第 10—15 月自行研习六个月只能完成一半，不能视为完全掌握；归元真人亲授六个月时可以完全掌握。残卷缺失比例的具体计算公式尚未决定。

### B/C 后：同强度赤潮兽灾

第二次赤潮仍公开检查三个具体问题，不使用宗门总战力：赤鳞妖王需要足以正面对抗的强者；赤鳞妖群需要护山阵与普通防务；反攻妖巢还需要第二名强者、适配法宝或其他机动追击能力。

- 岳徊路线：完整护山阵与稳定资源通路让守山、保基业获得明确改善，但林砚只自行学习六个月、尚未完全掌握《归元剑典》，不能把他按完整功法主战者计入；缺少专业法宝，反攻仍有缺口。
- 沈砺路线：强化护山阵与镇潮法宝形成建设和装备优势，但林砚只自行学习六个月、尚未完全掌握《归元剑典》，因此不能默认三个问题都已充分准备，也不自动进入压倒性优势档。
- 归元路线：归元、林砚与完整护山阵覆盖三个问题，对同强度旧赤潮形成两代强者并肩作战的压倒性优势；突破成功分支下归元在第 18 月大考后仍然在世，但长期资源通路与专业生产缺口仍然存在。

当玩家选择的目标所对应的三个问题都已有充分准备、形成压制优势时，旧赤潮进入“压倒性优势”公开档：守山门仍只结算 C（玩家主动放弃外部目标）且 100% 保住山门；保基业 100% 结算 B；反攻妖巢 90% A / 10% B / 0% C / 0% D。该档只表示宗门对这一次同强度旧问题准备充分；最终取得 A 才算彻底解决本轮兽潮，B/C 仍只是压制，兽潮此后仍会重新集聚。

压倒性优势下，每名具名参战者的伤势表暂用：全力进攻 75% 无伤 / 23% 受伤 / 2% 重伤 / 0% 死亡；攻守兼备 90% 无伤 / 9% 受伤 / 1% 重伤 / 0% 死亡；全力防守不选择反攻，其人物安全沿用或优于第一次大考的全力防守 B 档。具体比例当前可用于闭环纸面原型，必须在真人试玩后确认或调整，不升级为最终平衡。

### A 后：更强且不同类型的新兽潮（闭环纸面实例，非最终设定）

为使分支可执行，第二轮暂用“玄羽兽潮”作为纸面实例。玄羽兽王、裂空妖群、迁徙母巢是提前公开的三个问题，而不是三个被准备项自动永久消灭的目标。玄羽兽王的正面对抗门槛比赤鳞妖王高一档；裂空妖群能越过原本只针对地面的普通阵势，完整旧护山阵只能提供部分作用；迁徙母巢持续移动，追击需要第二名机动强者、适配法宝或稳定补给中的组合条件。

取得第一次 A 后，第 7—15 月没有旧赤潮重新集聚；宗门立即获得三层爆发奖励。第 9 月交接卡同时公开玄羽兽潮将在第 18 月形成，并公开上述三个新检查，让玩家能用 A 的资源提前准备，不以隐藏规则突然加难。

新兽潮的数值门槛更高，但既有成果不会失效：强者仍贡献正面对抗，旧阵仍提供部分防务，镇潮法宝仍可改造，稳定资源通路仍能提供持续补给。经营成果针对三个公开问题形成的准备程度，用来判断宗门能否压制兽潮并映射为防务完备、一处缺口、两处缺口或濒临崩溃，再使用已有 A/B/C/D 公开矩阵；本轮不另造第二套隐藏总战力公式。无论准备多充分，只有最终结算 A 才算彻底解决本轮玄羽兽潮；B/C 只是压制，兽潮仍会重新集聚。
