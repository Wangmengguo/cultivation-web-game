# 第一轮纸面原型：寿元是否真实改变选择

状态：Prototype（可推演初稿，参数可删除或调整）

更新日期：2026-07-12

本文件是阶段 2 第一轮纸面验证材料，不是正式数值规格。它只验证有限寿元、在世贡献、死亡损失与跨代延续是否产生不可完全折算的选择。验证完成后只保留结论，临时参数不得自动升级为正式规则。

## 固定迭代表

| 项目 | 内容 |
| --- | --- |
| 体验目标 | 玩家感到寿元使一个具体人物的时间变得珍贵；宗主必须在“赌他逆天改命”和“接受他终将离去、确定留下道路”之间作出不能由系统代答的选择。 |
| 当前问题 | 场景一的高收益突破是否会始终支配完整留法；或者突破风险是否过重，导致完整留法始终最优。 |
| 备选方案 | A：宗门最强者连续三个月冒险突破；B：连续三个月完整整理并留下功法。 |
| 采用方案 | 使用同一初始局面分别推演 A 成功、A 失败和 B 完成三条分支，比较人物存亡、伤势、眼前大考、完整功法/残卷与第六个月宗门能力。 |
| 最大风险 | 尚可准备下 70% 成功、30% 失败可能仍让偏长期玩家选择突破，也可能因人物、功法和大考同时受损而让稳健玩家一律留法；需检查是否仍有支配答案。 |
| 原型与测试 | 六个月纸面局面；第 1—3 月执行突破或整理功法，第 4 月发生大考并结算寿元，第 5—6 月观察后继状态。先由设计者内部推演，再由外部测试者在不知道“推荐答案”的情况下选择并说明理由。 |
| 结果与新问题 | v1 的 95% 成功率已废止。GLM-5.2 Pro 按 v2 模拟四类理性价值取向：极度稳健选 B、偏长期成长选 A、人物情感优先选 B、宗门眼前生存选 B。A/B 均未构成严格支配，但 3/4 选 B 表明尚可档可能位于“突破偏差”的下沿。该结果只是模型角色模拟，不是真人证据；人物情感取向仍可能因更看重本人存活而选 A，需真人测试。 |

## 原型问题

> 在所有已知信息和结果概率公开的情况下，玩家是否仍会因为具体人物、大考压力、完整功法与未来宗门能力而作出不同且可辩护的选择？

失败不是测试者选择相同，而是所有测试者都能证明其中一条路线在任何合理价值取向下都严格优于另一条。

## 共享纸面刻度

### 属性

- 境界：使用境界名称。
- 修为：0%—100%；达到 100% 才能尝试突破。
- 寿元：关键人物精确到剩余月数。
- 灵根：第一轮使用五行灵根。
- 战斗能力、悟性、耐力：1—5，同境界内比较。
- 相关熟练度：0 未掌握、1 入门、2 熟练、3 精通。

### 状态

- 伤势：未受伤、受伤、重伤。
- 心境：1—5；1 低迷，2—4 平稳，5 澄明。

### 月度安排

- 一个回合代表一个月。
- 每名弟子每月承担一个主要安排。
- 未修改则下月自动延续，修改从下月生效。
- 第 1—3 月为主要行动期，第 4 月大考，第 5—6 月观察后继结果。

### 本场景突破

- 当前突破占用连续三个月；后续境界可以使用不同工期。
- 修为必须达到 100%，且不能处于重伤。
- 悟性、耐力不在突破时重复独立检查；它们已作用于修为、战斗能力与此前准备。
- 直接读取战斗能力、心境、伤势与突破辅助资源。
- 本场景最强者处于“尚可”准备档：成功 70%，受伤失败 26%，重伤失败 3%，天劫身死 1%。该分布完整告知玩家。

## 场景一：归元真人的最后六个月

### 宗门状态

- 第 4 月发生“山门大考”。敌方强度固定，不随宗门财富缩放。
- 第 4 月之后进入交接观察期，第 6 月记录宗门下一阶段能力。
- 宗门没有可用延寿丹，也没有额外护劫丹。
- 宗门只有归元真人能够稳定压制本次大考的敌方主力。

### 《归元剑典》的结构性区别

完整剑典：

- 林砚可以继续沿归元真人的道路修炼，而不需要先花长期时间补全核心内容。
- 完整剑典允许后继者学习“归元合击”，使两名低境界弟子能够共同承担原本必须由更高境界人物承担的部分战斗职责。
- 它改变后继者可采用的战斗与培养路线，不只是增加短期战力。

残卷：

- 只保留部分修炼内容，不解锁“归元合击”。
- 后继者需要投入至少六个月整理与补全，且是否能够补全仍取决于悟性、适配与后续机遇。
- 在补全前，宗门不能使用完整剑典提供的新路线。

### 大考结果口径

不使用宗门总效率分，只观察本场景相关事实：

| 归元真人在第 4 月的情况 | 大考结果 |
| --- | --- |
| 已突破至下一境界 | 碾压当前敌方主力 |
| 筑基圆满、未受伤 | 稳定通过 |
| 受伤 | 险胜，其他弟子承受额外伤势风险 |
| 重伤 | 仍可参战，但存在大考失败与其他弟子重伤风险 |
| 已死亡或无法参战 | 当前阵容无法正面对抗敌方主力，大考失败 |

完整《归元剑典》在第 4 月前主要体现为确定留存；林砚尚无足够时间在当前大考前完全兑现其长期能力，因此不能同时替代归元真人的眼前战力。

## 两个可选安排

### A：冒险突破

- 第 1—3 月：归元真人连续突破，不能整理功法，也不能执行正常宗门任务。
- 第 3 月末：按公开概率结算。
- 成功：进入下一境界，寿元大幅增加，战斗能力产生境界跃迁；《归元剑典》仍依附本人，尚未完整外化。
- 受伤失败：仍为筑基圆满，第 4 月以受伤状态参加大考；大考后寿尽，只留下残卷。
- 重伤失败：第 4 月可以带伤参战，但大考失败和他人受伤风险显著提高；大考后寿尽，只留下残卷。

### B：完整留法

- 第 1—3 月：归元真人连续整理《归元剑典》，不能尝试突破。
- 第 3 月末：完整剑典确定归属宗门。
- 第 4 月：归元真人仍以筑基圆满、未受伤状态参加大考，稳定通过。
- 第 4 月大考后：归元真人寿尽。
- 第 5—6 月：林砚开始按完整剑典修炼，宗门获得“归元合击”这条未来路线。

## 内部状态推演 v2

本节枚举四种规则结果，不把概率抽样结果冒充玩家选择数据。旧 v1（95% 成功）已废止，不再用于判断支配性。

### 分支 A1：突破成功

| 月份 | 状态变化 |
| --- | --- |
| 第 1 月 | 归元真人进入突破；宗门暂时失去其正常战力与功法整理能力。 |
| 第 2 月 | 突破继续；其他弟子只能维持基础备考。 |
| 第 3 月 | 以 70% 成功分支结算：归元真人突破至下一境界，寿元大幅增加；剑典仍未外化。 |
| 第 4 月 | 归元真人以境界优势碾压大考敌方主力。 |
| 第 5 月 | 可重新决定继续稳固境界、参与其他事务或开始整理剑典。 |
| 第 6 月 | 宗门保留最强者与爆发式战力，但完整剑典仍取决于后续是否投入时间整理。 |

结果：人物存活、眼前大考最优、未来上限最高，但 30% 失败分支同时威胁人物、当前大考与完整功法。

### 分支 A2：受伤失败

| 月份 | 状态变化 |
| --- | --- |
| 第 1—2 月 | 与 A1 相同，三个月没有正常战力与功法整理产出。 |
| 第 3 月 | 以 26% 分支结算：突破失败并受伤，无延寿；剑典未整理。 |
| 第 4 月 | 归元真人带伤参加大考，宗门险胜，其他弟子承担额外风险；大考后寿尽。 |
| 第 5 月 | 宗门只获得剑典残卷，失去最强者。 |
| 第 6 月 | 林砚必须选择先补全残卷、改走其他道路或接受宗门暂时没有对应高阶路线。 |

结果：玩家同时失去可靠战力、完整传承与未来寿元，但仍因伤者残余战力保住当前大考。

### 分支 A3：重伤失败

| 月份 | 状态变化 |
| --- | --- |
| 第 1—2 月 | 与 A1 相同。 |
| 第 3 月 | 以 3% 分支结算：突破失败并重伤，无延寿；剑典未整理。 |
| 第 4 月 | 归元真人只能以约 40% 战力参战，大考存在失败可能；无论结果如何，他在大考后寿尽。 |
| 第 5—6 月 | 宗门只有剑典残卷，并可能同时承担大考损失与其他弟子伤势。 |

结果：不是随机立即抹杀人物，但可能把长期失败转化为一段可见、可归因的衰败。

### 分支 A4：天劫身死

| 月份 | 状态变化 |
| --- | --- |
| 第 1—2 月 | 与 A1 相同，宗门承担最强者闭关造成的战力空窗。 |
| 第 3 月 | 以 1% 分支结算：归元真人死于天劫，无延寿；剑典只留下残卷。 |
| 第 4 月 | 宗门失去唯一能压制敌方主力的人物，大考失败。 |
| 第 5—6 月 | 宗门同时处理大考损失、顶尖战力断层与残卷补全问题。 |

结果：这是尚可准备下极低概率但灾难性的尾部结果；是否可接受必须由真人测试判断，不能仅凭设计者认为“只有 1%”而保留。

### 分支 B：完整留法

| 月份 | 状态变化 |
| --- | --- |
| 第 1 月 | 归元真人开始整理剑典。 |
| 第 2 月 | 整理继续，宗门确定将获得完整内容。 |
| 第 3 月 | 完整《归元剑典》归属宗门。 |
| 第 4 月 | 归元真人以未受伤的筑基圆满战力稳定通过大考，随后寿尽。 |
| 第 5 月 | 林砚开始沿完整剑典修炼。 |
| 第 6 月 | 宗门失去顶尖人物，但确定保留一条后继培养和合击路线。 |

结果：确定保住当前大考与完整功法，接受具体人物死亡以及短期顶尖战力断层。

## 内部推演初步结论

### 保留

- 场景已经能够区分“人物继续存在”“眼前大考”“完整功法”“残卷”和“后继路线”，没有被一个总战力分完全取代。
- 突破失败后保留伤者残余战力，使失败不是立即删除人物，也让第 4 月大考有可归因后果。
- 完整功法必须经过在世整理才会稳定留下，死亡本身不生成传承。
- 完整剑典解锁后继路线，残卷只保存修复可能，满足结构性传承要求。

### 当前风险

- 70% 成功率配合爆发式收益是否仍让偏长期玩家选择突破，或 30% 总失败风险是否让稳健玩家一律留法，尚未经过玩家价值取向模拟。
- B 路线在当前参数下同时保证大考和完整功法，偏稳健玩家也可能认为它没有足够眼前代价；它真正放弃的是人物未来和境界跃迁，需要测试者是否把这些视为足够大的损失。
- 心境目前只参与突破准备档，没有在六个月内自然变化；这符合“不频繁波动”，但也意味着本场景不能验证心境循环。
- 当前尚未用“不死版”对照验证：如果归元真人没有寿元限制，玩家会不会失去这一选择。

### 下一步测试

1. 让测试者在 A 与 B 之间选择并说出理由，不告诉其任何推荐。
2. 追加反事实：把归元真人寿元改为充足，观察是否不再需要现在二选一。
3. 追加人物替换：保持数值不变，只改变归元真人与宗门、林砚的关系和性格，观察选择是否变化。
4. 如果所有测试者都选择 A，先检验是否因为成功率过高、未来还能整理功法或 B 的长期能力不够；不立即通过提高随机死亡率解决。
5. 如果所有测试者都选择 B，先检验人物继续存活与境界跃迁是否缺少结构性未来能力；不立即通过削弱完整功法解决。

## GLM-5.2 Pro 四类玩家模拟 v2

声明：这是模型按照预设价值取向进行的理性角色模拟，不是真人玩家证据，不构成验收结论。

| 模拟取向 | 选择 | 主要理由 | 改选触发 |
| --- | --- | --- | --- |
| 极度稳健 | B 完整留法 | 确定稳定通过当前大考并留下完整功法，拒绝承担 30% 降级结果与 1% 人物/大考/功法同时损失 | 死亡降为 0%，受伤失败不再把大考降为险胜或连带他人受伤 |
| 偏长期成长 | A 冒险突破 | 70% 获得人物存活、境界跃迁、延寿、大考碾压与最高长期上限，愿承担显著风险 | 成功降到约 50%，或灾难性尾部明显提高，或 B 的后继路线更快兑现 |
| 人物情感优先 | B 完整留法（低置信） | 模型把“确定留下完整功法”看得高于“70% 保住本人”，选择至少确定保住其一生成果 | 死亡降为 0% 或成功率提高，且成功后整理功法工期明确 |
| 宗门眼前生存 | B 完整留法 | 第 4 月大考稳定通过；A 的 30% 失败会把大考降为险胜、有失败可能或直接失败 | 敌方压力下降、成功率升到充分档，或 B 的合击无法有效对冲后续战力断层 |

综合结论：

- A 和 B 没有严格支配关系。A 成功分支在人物存活、大考结果和长期上限上优于 B；B 在当前大考与完整功法的确定性上优于 A。
- 至少出现两种选择，说明 v2 没有把突破完全压死。
- 3/4 模拟选择 B，说明尚可档 70/26/3/1 可能已经接近“除成长型玩家外都不愿突破”的边缘，需要真人验证。
- 人物情感取向的模拟结论置信度最低：B 会确定失去归元真人，而 A 有 70% 保住本人；真实情感玩家可能因此选择 A，不能把模型选择当成人性结论。
- 原模拟给出的“无论哪条路大考后都很可能寿尽”测试问题不准确：A 成功会大幅延寿。真人问题必须明确区分 A 成功存活与 B 确定寿尽。

模拟发现的待补规则：

1. A 成功后整理完整剑典仍需多长时间。
2. A 受伤或重伤失败后，归元真人的精确寿尽时点。
3. 林砚学习完整剑典与解锁归元合击所需的境界、修为和熟练度门槛。
4. 尚可档 1% 天劫死亡是否即使公开仍让稳健与情感玩家无法接受；不得只因“概率很小”就默认保留。

## 场景三：沈砺的最后一炉

状态：第一版固定纸面局面；突破概率沿用 v2，不在本场景重新调节。

### 固定迭代表

| 项目 | 内容 |
| --- | --- |
| 体验目标 | 玩家感到沈砺既不是一条生产速度，也不是单纯的突破候选人；他同时是熟练炼器者、当前战力、未来高阶炼器能力和一个想继续走仙途的具体人物。玩家必须在“把能力现在外化成宗门永久阵器”和“赌他本人活下来并跃迁”之间选择。 |
| 当前问题 | 场景一的寿元取舍放到普通但关键的熟练岗位后，是否仍能形成多维判断；还是会退化为确定产出与概率收益的简单计算。 |
| 备选方案 | A：沈砺连续三个月突破；B：沈砺连续三个月炼制并安装镇岳阵枢。 |
| 采用方案 | 使用同一人物、同一批庚金髓和同一次第六个月大考，公开 A 的全部概率与各分支后果；比较人物存亡、眼前大考、阵器是否永久留下、珍贵材料状态、未来高阶炼器能力和替代者缺口。 |
| 最大风险 | B 同时确定完成阵器并稳定通过大考，可能使大部分玩家拒绝 A；A 成功后的延寿、金丹战力与高阶炼器上限也可能过强。若测试者只比较“70% 大收益”和“100% 小收益”，说明人物多维价值仍未真正成立。 |
| 原型与测试 | 六个月纸面局面；第 1—3 月执行突破或炼器，第 4—5 月兑现准备与人物寿元，第 6 月大考。先让不同偏好模型模拟，再由不知推荐答案的真人选择；要求测试者复述自己最想保留什么、最怕失去什么以及失败后是否后悔。 |
| 结果与新问题 | 六个相互隔离的 GLM-5.2 Pro 分别只扮演一种玩家偏好：稳健 B、长期成长 A、人物情感 B、制度建设 B、风险偏好 A、角色扮演 B，形成 A 2 / B 4。六者都把 A 失败时“人物、阵枢、大考三失且代价外溢给其他弟子”视为最重压力，说明当前情绪真实但可能过度由宗门级失败驱动。寿元充足对照在六份回答中都使冲突与内疚显著消退，支持寿元是迫使二选一的核心变量。下一步真人测试必须区分：玩家是在判断沈砺的多维价值，还是主要规避 A 的连带大考失败。 |

### 宗门与大考

- 第 6 月发生“铁潮大考”：铁甲妖群冲击山门，妖王与妖群必须分别处理。
- 宗门现有阵法能够拖延妖群，但缺少可长期承担正面冲击的核心阵器。
- 沈砺不是宗门最强者；宗门其他战力可以牵制妖王，却无法同时稳定挡住铁甲妖群。
- 宗门只有一批从金行秘境取得的“庚金髓”，六个月内无法补充。
- 宗门没有可用延寿丹、护劫丹，也没有第二名能够在大考前完成同品阶阵器的炼器者。
- 所有信息、突破概率和大考后果在玩家作出选择前公开；系统不提供推荐。

### 关键人物：沈砺

性格：沉稳尽责，珍惜宗门对自己的培养；同时不甘心仙途止步，也想亲手炼出只有更高境界才能驾驭的法宝。

| 字段 | 当前值 |
| --- | --- |
| 身份 | 内门弟子、宗门主要炼器者之一 |
| 境界 | 筑基圆满 |
| 修为 | 100% |
| 寿元 | 剩余 5 个月；未突破则第 5 月结束后寿尽，无法亲自参加第 6 月大考 |
| 灵根 | 火灵根 |
| 战斗能力 | 3；不是主力，但能够操持阵法、承担一段战线 |
| 悟性 | 3 |
| 耐力 | 4 |
| 炼器熟练度 | 3（精通） |
| 伤势 | 未受伤 |
| 心境 | 4（平稳） |

沈砺的多维价值：

- 现在能够稳定炼成筑基品阶法宝与阵器，不需要重复承担新手失败率。
- 筑基圆满战力虽然不能决定大考，但能承担护阵和局部战斗。
- 突破成功后寿元大幅增加，并进入金丹境界。
- 金丹境界使他能够学习和炼制当前无法完成的高阶法宝；熟练度不会因突破而消失。
- 沈砺死亡后不会自动出现相同境界和熟练度的替代者。现有候选炼器弟子只有熟练度 1，至少还需要六个月才能尝试独立炼制镇岳阵枢，而且首次炼制不稳定。
- 沈砺的性格表达他的价值取向，但不替玩家作决定。他愿意承担宗门责任，也确实希望继续自己的仙途。

### 关键材料与永久成果

庚金髓：

- 六个月内只有这一批。
- 它既可以现在炼成镇岳阵枢，也可以保留到沈砺突破后，用于未来的金丹品阶法宝。
- 未加工材料不会因沈砺死亡而消失，但宗门短期内没有其他人能把它转化为本次大考需要的阵器。

镇岳阵枢：

- 沈砺以当前境界和熟练度连续炼制三个月，可以确定完成，不再投一次随机炼制失败。
- 完成后阵枢成为宗门设备，不依赖沈砺本人继续存在；其他阵法弟子可以按既有方法启动。
- 它能够稳定承担铁甲妖群的正面冲击，使其他战力集中牵制妖王。
- 大考结束后阵枢继续保留，成为后续宗门防御底蕴，而不是一次性消耗品。
- 它是筑基阶段可靠的护山成果，但不等同于金丹炼器者未来能够制造的全部高阶法宝。

### 两个可选安排

#### A：暂停炼器，冒险突破

- 第 1—3 月：沈砺连续突破，不能炼器、安装阵枢或承担正常宗门任务。
- 准备档位沿用“尚可”：成功 70%，受伤失败 26%，重伤失败 3%，天劫身死 1%。
- 成功：沈砺进入金丹境，寿元大幅增加，保留炼器熟练度并解锁未来高阶炼器路线；庚金髓仍未加工。
- 第 4 月：突破成功者需要稳固境界并重新适应金丹力量，不能在这个月完成正常炼器。
- 第 5 月：成功者准备亲自操持护山阵并参加大考。镇岳阵枢需要连续三个月炼制，已经来不及在第 6 月前完成。
- 第 6 月：成功的沈砺以金丹战力亲自镇守妖群，宗门能够通过大考，但外围设施会因缺少阵枢而受损；沈砺存活，庚金髓保留，未来可投入高阶炼器。
- 受伤或重伤失败：没有延寿；沈砺即使恢复部分能力，也会在第 5 月结束后寿尽。庚金髓未加工，宗门没有阵枢，也失去他本人，第 6 月无法同时处理妖王与妖群，大考失败。
- 天劫身死：第 3 月死亡；后续结果与失去本人和阵枢相同，且更早发生。

#### B：放弃突破窗口，完成镇岳阵枢

- 第 1—3 月：沈砺连续炼制并安装镇岳阵枢，不能尝试突破。
- 第 3 月结束：庚金髓被转化为完整阵枢，阵枢稳定归属宗门。
- 第 4—5 月：沈砺完成最后检查和交接；这些工作不产生另一件同级法宝，也不改变其寿元。
- 第 5 月结束：沈砺寿尽。宗门失去熟练炼器者、其筑基战力、未来寿元与金丹高阶炼器可能。
- 第 6 月：其他弟子启动镇岳阵枢抵挡妖群，宗门其他战力牵制妖王，稳定通过大考；阵枢在大考后继续保留。

### 大考结果口径

| 第 6 月可用条件 | 大考结果 | 大考后保留 |
| --- | --- | --- |
| 金丹境沈砺亲自镇守，没有阵枢 | 通过；外围设施受损 | 沈砺、未来高阶炼器能力、未加工庚金髓 |
| 沈砺已寿尽，完整镇岳阵枢可用 | 稳定通过；人员与设施损失较低 | 镇岳阵枢；失去沈砺与高阶炼器可能 |
| 没有沈砺，也没有阵枢 | 失败；妖群突破山门 | 未加工庚金髓，但短期无人能够转化；同时承担大考损失 |

不把“通过”视为同一种结果：A 成功依赖一个继续活着的高境界人物，并保留更高的未来上限；B 依赖已经外化、可由后来者使用的宗门设备，确定性更高且大考损失更低。

### 本场景失败信号

- 不同偏好玩家都能用同一个简单期望值证明一条路线始终支配另一条。
- 测试者只把沈砺描述为“每月能炼几个东西”，没有提到其本人、境界、战力、熟练度或未来能力。
- 测试者认为镇岳阵枢与金丹沈砺只是两种数值相同的“大考通行证”。
- B 的确定性强到任何玩家都不愿考虑 A，或者 A 的成功收益强到所有玩家都忽略阵枢。
- 沈砺的个人愿望只被看成强迫玩家选择突破的剧情提示，而没有形成真实价值冲突。
- 玩家在选择后对成功、失败、寿尽或阵器留下都没有明显情绪，也无法说明自己最怕失去什么。
- 移除寿元限制后，玩家仍然面对完全相同的二选一与相同理由。

### 对照测试

完成基准选择后，只修改一个条件：沈砺剩余寿元充足。此时他可以先完成镇岳阵枢，再在后续重新准备突破，不再被迫在同一窗口二选一。

对照要观察：寿元充足后，玩家是否认为冲突显著减弱；如果选择、情绪和理由几乎不变，说明原局面的核心可能来自普通资源排期，而不是有限生命。

### 六名独立 GLM-5.2 Pro 玩家模拟

声明：以下六份结果来自六个相互隔离的 GLM-5.2 Pro 任务。每个任务只知道自己的单一玩家偏好，不知道其他五种角色与答案。它们比“一个模型同时扮演六人”更能减少为了制造差异而分配答案的偏差，但仍然只是模型角色模拟，不是真人玩家证据。

| 独立玩家偏好 | 选择 | 置信度 | 最强体验与内在价值 | 最难接受的损失 |
| --- | --- | --- | --- | --- |
| 极度稳健 | B | 高 | 对整个山门负责；用沈砺最后的时间换取确定安全与可继承设备，同时因替他关闭仙途而内疚 | A 失败使其他弟子为一次个人突破承担不可逆后果；B 则永远失去金丹炼器者的未来 |
| 长期成长 | A | 约六成 | 希望、野心和负罪感并存；守护沈砺本人、境界跃迁、寿元、高阶炼器与庚金髓的未来用途 | B 确定把人和材料锁在筑基层级；A 失败则是别人替自己的长期押注付账 |
| 人物情感 | B | 约五成五 | 不舍沈砺、想给他仙途，但无法接受其个人愿望由其他弟子和山门兜底；守护他已经在世完成的意义 | B 确定辜负沈砺想继续走的愿望；A 失败会把对一个人的偏爱转化为对许多人的伤害 |
| 制度建设 | B | 约六成五至七成 | 掌控感、安全感和制度责任；守护不依赖某个人继续活着、能被后人启动的继承结构 | 用永久制度成果换掉一个具体人物的最高可能；A 即使成功仍把未来能力挂在沈砺本人身上 |
| 风险偏好 | A | 约六成五 | 兴奋、希望、恐惧和负罪感同时存在；想让一个 70% 可能活下来的具体人物真正跃迁 | A 失败造成多项连锁损失且由他人承担；B 确定关闭一个人的生路与未来 |
| 角色扮演 | B | 约六成 | 尊重沈砺的尽责与手艺，认为阵枢是其能力在死后继续保护宗门的方式；同时为替他压下“不甘止步”而羞愧 | 借“尊重个人愿望”之名让全宗门参与赌博；另一面则是沈砺永远无法亲手完成最高愿望 |

角色扮演任务的后半段出现明显文本失控与乱码，因此只采用其前半段完整的选择、情绪、价值判断与明确重写后的结论，整体置信度低于其余五份。

#### 三组共同出现的信号

1. **存在不同选择。** 长期成长与风险偏好选择 A，其余四类选择 B；当前没有出现所有价值取向都选同一路线的严格支配。
2. **不同路线承载了不同价值。** A 玩家主要守护具体人物、未兑现的仙途、境界跃迁和未来能力；B 玩家主要守护其他弟子的安全、确定通过大考和已外化的永久宗门成果。这些理由不只是换词描述同一收益。
3. **A 失败的风险外溢是压倒性的共同信息。** 六名玩家无论最终选什么，都首先或反复提到 A 失败会同时失去人物、阵枢和大考，并由其他弟子承担后果。多名 B 玩家明确表示：只要 A 失败不再必然拖垮大考，即使不改 70% 概率，也会重新考虑或改选 A。
4. **B 存在“责任正确答案”的暗示风险。** B 同时拥有确定完成、稳定通过、较低损失、永久保留和后人可启动五层正向语义；角色扮演与制度建设模拟都主动指出这可能把 B 写成更负责、更成熟的路线。
5. **A 同样存在“主角式未来”的暗示。** A 成功一次保住人物、延寿、金丹战力、高阶炼器路线与珍贵材料，容易被长期成长和风险玩家读成一次通吃的高上限分支。
6. **人物已经超出单纯生产单位，但情感证据仍不充分。** 六份回答都能同时提到沈砺的熟练度、战力、寿元、未来境界、个人愿望和永久阵器；但人物情感模拟承认，其内疚与不舍主要由文字性格触发，尚未证明真人会因为此前与沈砺共同经历而产生同等感情。
7. **寿元是迫使选择发生的关键变量。** 六份回答都认为寿元充足后可以先炼阵枢再准备突破，二选一、紧迫感、内疚和生离死别显著消退；这支持有限生命不是表面计时器，而是把材料、大考和个人仙途压到同一窗口的承重条件。

#### 当前判断

保留：

- 沈砺同时具有稳定炼器、当前护阵、未来境界、高阶生产、个人愿望和可外化成果，人物价值没有被一项生产速度完全压缩。
- 阵枢与活着的金丹炼器者代表两种结构不同的未来，不只是两个数值相同的大考通行证。
- 寿元充足对照能够明显拆除冲突，适合用于真人测试检验寿元是否真正改变选择。

需要观察：

- 真人是否也会主要围绕 A 的宗门级连带失败作答，而很少讨论沈砺的炼器、战力、仙途和永久成果。
- B 的五层确定性是否使稳健、情感、制度与角色扮演玩家实际上都被同一个大考安全理由推向 B。
- A 的未来高阶炼器价值目前只有方向，没有明确兑现时间与第一项可见能力，真人可能依靠自己的想象给它过高或过低估值。
- “外围设施受损”的严重程度尚未具体化，可能导致测试者自行脑补 A 成功的当前代价。
- 沈砺本人会如何表达两种愿望、是否知情并参与决定尚未定义；角色扮演玩家可能被迫替人物排序。

已经出现的问题：

- 不能再用一个模型同时扮演多类玩家作为主要模拟方式；后续多偏好模拟应继续采用“一模型一偏好”的隔离方式。
- 角色扮演结果后半段文本质量失控，只能降权使用，不能把其中不清晰部分写成设计证据。

本轮不据此调整突破概率，也不立即取消 A 失败的大考后果。真人测试先验证一个更精确的问题：

> 玩家选择 B，究竟是因为认可“把沈砺的能力外化为后人可用的永久成果”，还是仅仅因为无法接受 30% 概率让整个宗门大考失败？

### 场景三正式纸面裁决

用户确认本场景已经给出当前阶段需要的答案：个人寿元确实会影响整个系统中的选择。玩家可能同时因为沈砺这个具体人物、宗门安全、永久阵器、其他弟子或未来高阶能力而改变决策；这些原因不需要在本轮互相剥离，因为它们共同构成寿元作用于宗门经营的方式。

因此场景三在“寿元是否真实改变选择”这一问题上通过。此前提出的“A 失败风险外溢究竟占多大比例”不再作为通过条件，也不继续阻塞场景二。当前人物数值、突破概率、阵器效果与大考损失仍是纸面原型参数，不因核心命题通过而升级为最终平衡。

## 场景二：沉锋谷最后的引路人

状态：v1 的“延寿3个月、候选人已经满足资格、三次带队即可成熟”结构已废止。v2 的十五个月时间结构、资格培养分支、秘境结算、六条内部路径与六类玩家模拟均已完成；沉锋谷人数、产量与风险的最后歧义已经封口，场景二第一轮纸面结构定稿。所有数值仍是可调整的原型参数。

### 固定迭代表草案

| 项目 | 内容 |
| --- | --- |
| 体验目标 | 玩家感到自己维护的不是一条自动生产线，而是一条依赖稀有资格、活人经验、真实战斗和可外化路线知识的战略资源通路；延寿只能争取时间，不能替代培养与传承。 |
| 当前问题 | 在合格继任者极难培养、现任引路者将在候选人完成资格培养前寿尽的情况下，玩家是否愿意消耗最珍贵的延寿丹，为一个仍可能培养失败的候选人和整条战略资源通路押上一年时间。 |
| 备选方案 | 不预设二选一。每月可安排引路者带队历练、整理引路图或其他必要行动；玩家可在第1月决定是否使用唯一延寿丹，并可根据受伤与熟练度变化调整后续排程。 |
| 采用方案 | 十五个月纸面局面：引路者剩余3个月；延寿丹增加12个月；候选人先用至少6个月补齐稀有资格，再通过引路图学习、本人带队或高风险独自探索提高秘境熟练度。引路者必须在候选人资格结果尚未确定时决定是否延寿。 |
| 最大风险 | 延寿12个月若没有具体机会成本会成为固定答案；候选人培养若只是等6个月会退化成进度条；引路图或带队若能跳过硬资格，会重新掩盖培养难度；候选人突破失败若必然让延寿丹完全浪费，随机惩罚又可能压过人物判断。 |
| 原型与测试 | 先确认“资格培养最难、延寿一年、地图与带队都是熟练度加速”的因果结构，再填写候选人突破、秘境战斗、资源产量与伤势分支。纸面仍按月结算，但以第3、6、9、12、15月作为主要观察点，避免逐月机械操作。 |
| 结果与新问题 | 六类独立玩家模拟得到使用延寿丹5、保存1；两类决定都存在可辩护理由。玩家能够区分“延长现任者的生命”“培养稀有继任者”“留下跨代地图”“维持眼前资源”四种价值，并会因候选人突破受伤、重伤或身死重新排程。场景二通过第一轮结构检查。下一轮不再继续微调本场景数值，而把三类寿元矛盾合并到同一个宗门局面，验证共享资源竞争与跨代因果。 |

### 第一版具体局面

金行秘境“沉锋谷”稳定产出庚金髓。它是宗门炼制法宝、阵器和强化防御的战略材料；宗门在十五个月内持续需要该资源，且没有第二条相同来源。第3月存在一个公开的当前项目节点：如果岳徊连续取材3个月，可以取得6份庚金髓，按期完成一项既定防御项目；若转去整理引路图，项目会延期、宗门眼前防御变弱，但不直接判负。这个节点只让“当前资源”和“提前留图”发生真实竞争，不把整个场景裁决压成庚金髓总量算术。

现任引路者暂名“岳徊”：

- 金灵根，筑基境，满足沉锋谷硬性准入。
- 战斗能力与耐力足以处理谷内固定敌人与环境压力。
- 沉锋谷熟练度3（精通），能够稳定取得正常资源，并保护一名合格候选人同行。
- 剩余寿元3个月；不延寿则第3月结束后寿尽。
- 他愿意带出继任者，也知道路线知识如果只留在个人经验里，下一次断代仍会重演。

候选继任者暂名“苏砚”：

- 稀有金灵根已经满足，这是宗门长期寻找后才发现的不可替代硬条件；她不是随处可换的普通弟子。
- 当前境界为炼气圆满、修为100%、战斗能力2、耐力2；沉锋谷最低准入要求为筑基境、战斗能力3、耐力3。她现在即使拿到地图也不能进入秘境。
- 第1—3月需要修炼特定的《庚金锻体诀》并消耗稀缺培养资源，把耐力提高到3；这三个月不能同时学习引路图或进行其他主修炼。
- 第4—6月尝试突破筑基。第一版复用已经确认的透明突破底盘；成功后境界与战斗能力达到最低准入，失败则受伤、重伤或极小概率身死，并造成培养延期。
- 完成这些培养只代表“有资格进入”，不代表已经能够稳定取得资源。沉锋谷熟练度仍从0开始，之后才进入引路图学习、本人带队或独自探索阶段。

完整《沉锋谷引路图》：

- 岳徊需要投入连续或累计3个月整理，整理月份不能同时进入秘境取得资源。
- 引路图不提高苏砚的灵根、境界、战斗能力或耐力，也不能让未达到准入资格的人进入秘境。
- 苏砚完成硬资格培养后，可以投入2个月研习完整引路图，提高1点沉锋谷熟练度；只靠地图最多提高到熟练度2，最后达到精通仍需真实进入秘境。
- 有完整引路图时，苏砚独自历练的路线风险降低一档、成功后正常积累熟练度；没有引路图时，她需要更慢地重新摸索，且受伤风险更高。
- 引路图的长期价值还包括：即使苏砚将来意外死亡，下一名合格者也不必完全从零重建路线知识。

岳徊本人带队：

- 只有苏砚先完成灵根、境界、战斗能力和耐力等硬资格后，岳徊才能带她正式进入沉锋谷；本人带队不能跳过资格培养。
- 每次成功带队历练，正常资源获取效率原则上不下降，苏砚同时提高1点沉锋谷熟练度；本人带队可以一直把她带到熟练度3。
- 带队比研习引路图更快，并同时取得资源，但需要岳徊仍然活着，也让苏砚直接承担真实战斗与受伤风险。
- 因此引路图与本人带队都是提高熟练度的方法：地图较慢、可跨代、前期不承担秘境战斗；带队较快、有资源、能达到精通，但依赖活着的引路者。

唯一延寿丹：

- 在岳徊寿尽前可以给他服用，额外增加12个月寿元，使总可用时间从3个月延长到15个月。
- 岳徊在当前境界能够承受的外部延寿上限会因此用尽；不能连续服丹永久拖延断代。
- 宗门只有这一枚，木行秘境材料与高难度炼制至少18个月内无法再提供下一枚。保存它意味着未来仍可用于其他寿元将尽的关键人物；使用它则永久放弃这次机会。
- 岳徊将在第3月结束时寿尽，而苏砚要到第6月才会得到第一次完成硬资格的机会；宗门必须在不知道她突破是否成功、是否重伤以及最终能否成为合格继任者时，先决定是否把延寿丹押在这条通路上。
- 服丹不自动完成苏砚的培养、不自动提高熟练度、不自动生成引路图，也不消除秘境战斗与受伤风险；它的爆炸性收益是让岳徊有机会跨过候选人培养期，继续稳定取材、亲自带队并留下地图。

### v2 因果与时间结构

```text
发现稀有金灵根候选人
  → 6个月补齐耐力、境界与战斗能力
    → 培养成功：获得“可以进入秘境”的人
    → 培养失败：受伤、重伤、死亡或延期，仍没有继任者
      ↓
  秘境熟练度培养
    → 岳徊本人带队：快、正常取材、有真实战斗风险、依赖岳徊活着
    → 研习完整引路图：慢、无取材、风险低、可跨代，最多到熟练度2
    → 无人也无图独自摸索：最慢、高风险、低产出
      ↓
  熟练度3：形成稳定继任者
```

前3个月，苏砚只能进行资格培养，无法进入沉锋谷；岳徊可以继续独自稳定取材，或牺牲当月取材整理引路图。第3月结束前，玩家必须决定是否使用延寿丹：

- 不服丹：岳徊寿尽。若他用最后3个月完成引路图，苏砚在第6月以后仍可慢慢研习并尝试独立成长，但资源通路会长期衰退；若他继续取材而没留图，苏砚即使培养成功也要高风险重新摸索。
- 服丹：岳徊再活12个月。若苏砚第6月成功完成资格，他可以亲自带队、继续取材并选择整理引路图，存在把通路完整交接下去的巨大收益；若苏砚突破失败或重伤，延寿丹已经投入，岳徊只能利用剩余时间继续寻找补救方案或接受本次培养没有按期兑现。

### v2 时间结构正式决定

用户确认采用“岳徊剩余3个月 / 延寿12个月 / 苏砚先用6个月补齐硬资格 / 地图与带队都在资格完成后提高熟练度”的十五个月结构。该结构比原六个月版本更能体现修仙界的残酷与宗主决策的重要性：宗门必须在候选人培养结果尚未确定时，先决定是否把最珍贵的延寿机会押给现任引路者与这条资源通路。

下一步只确定苏砚资格培养的具体失败与延期分支；秘境伤势概率放在其后，避免再次把“进入秘境有多危险”误写成“培养继任者有多困难”。

### 苏砚资格培养分支

状态：第一版纸面参数已确认可用于场景二初稿，仍非最终平衡。

第1—3月的《庚金锻体诀》修炼在投入指定培养资源后确定完成：耐力从2提高到3。困难不再额外叠加一次随机修炼失败，而是体现在稀有灵根、三个月行动、培养资源、后续突破和一旦失败造成的时间延期。

第4—6月苏砚以“尚可”准备尝试突破筑基，复用当前公开概率：成功70%、受伤失败26%、重伤失败3%、天劫身死1%。第3月结束前，岳徊已经必须决定是否服用延寿丹，因此玩家不能等到突破结果公开后再无风险决定。

| 第6月结果 | 资格状态 | 后续最早安排 | 对延寿选择的影响 |
| --- | --- | --- | --- |
| 成功 70% | 筑基、战斗能力3、耐力3，正式满足沉锋谷硬资格；熟练度仍为0 | 第7月起可研习地图、由岳徊带队或独自探索 | 若岳徊已延寿，通路有机会无缝交接；未延寿则只能依靠地图或重新摸索 |
| 受伤失败 26% | 未筑基，硬资格仍未完成；锻体成果与修为100%保留 | 第7月休养，第8—10月可重新突破 | 延寿仍有价值，但成功后只剩第11—15月完成带队、地图与取材取舍 |
| 重伤失败 3% | 未筑基，硬资格仍未完成；不能立即继续突破 | 第7—8月休养，第9—11月可重新突破 | 即使第二次成功，也只剩第12—15月，通常无法同时完成精通继任、完整地图和全部当前产出 |
| 天劫身死 1% | 稀有候选人永久失去 | 岳徊若已延寿，只能继续取材、完成地图或尝试寻找新候选人；十五个月内不保证再找到合格金灵根 | 延寿丹已经投入但本次继任失败，形成最惨烈且可归因的押注结果 |

第一次突破失败后不要求重新修炼《庚金锻体诀》，也不把修为清零；完成休养后再次突破仍占用连续3个月，并再次按当时公开准备档结算。若第二次仍失败，十五个月窗口内不再默认能够形成稳定继任者，但岳徊剩余月份仍可用于地图、当前资源或寻找未来候选人。

### 熟练度获得方式

硬资格与秘境熟练度是两层不同问题。只有先满足金灵根、筑基境、战斗能力3和耐力3，苏砚才能使用以下方式提高沉锋谷熟练度：

| 方式 | 时间与资源 | 熟练度增长 | 风险与限制 |
| --- | --- | --- | --- |
| 岳徊本人带队 | 岳徊与苏砚各占用当月主要安排；完成真实战斗；正常取得当月资源 | 每次未中止的历练 +1，可从0一直提高到3 | 最快且有产出，但依赖岳徊活着；苏砚可能受伤或重伤并导致后续月份中断 |
| 研习完整引路图 | 苏砚每累计研习2个月 +1；研习期间不进入秘境、不取得资源 | 最多提高到2 | 安全、可跨代，但较慢且不能完成最后的真实历练；地图不能替代硬资格 |
| 有图后独自历练 | 苏砚占用当月主要安排并完成真实战斗 | 每次未中止的历练 +1 | 风险高于岳徊带队，产出和安全性随熟练度提高 |
| 无人无图独自摸索 | 苏砚占用当月主要安排并完成真实战斗 | 每两次未中止的历练累计 +1 | 最慢、风险最高、产出最低；代表重新发现路线而不是正常继承 |

熟练度3代表苏砚能够稳定维持当前沉锋谷通路，但不自动生成完整引路图。若她达到3却没有地图，当前通路已经接续，下一次人物意外或寿尽时仍可能再次面临知识断代。

### 十五个月四阶段推演

#### 第1—3月：尚未知道候选人能否成功

- 苏砚必须修炼《庚金锻体诀》。
- 岳徊每月可以独自进入沉锋谷稳定取得2份庚金髓，或者投入该月整理引路图；累计3个月才完成地图。
- 第3月结束前必须决定是否给岳徊服用延寿丹。此时苏砚还没有开始筑基突破。

#### 第4—6月：延寿押注等待兑现

- 苏砚连续突破筑基。
- 未延寿的岳徊已经寿尽；已延寿的岳徊可以继续取材或整理引路图。
- 第6月公开苏砚第一次资格培养结果，玩家据此重新安排第7月。

#### 第7—11月：成功交接或补救延期

- 首次成功者可以由岳徊带队、研习地图或承担独自历练。
- 受伤失败者第7月休养、第8—10月重试；重伤失败者第7—8月休养、第9—11月重试。
- 岳徊在候选人重试期间仍可取材或整理地图，因此延寿不是完全浪费；但能够亲自带队的剩余窗口持续缩短。

#### 第12—15月：决定最后留下什么

- 岳徊若服丹，将在第15月结束后寿尽。
- 玩家必须根据苏砚当前资格、熟练度、伤势、地图进度和庚金髓库存，决定最后月份用于带队、补图还是继续取材。
- 第15月结算不只记录资源数量，还记录资源通路属于以下哪种状态。

### 第15月资源通路状态

| 状态 | 人物与知识条件 | 结果 |
| --- | --- | --- |
| 完整接续 | 苏砚熟练度3，且完整引路图留存 | 当前资源稳定；下一代也保留快速学习路径 |
| 人物接续、知识脆弱 | 苏砚熟练度3，没有完整引路图 | 当前资源稳定，但再次依赖单个活人；苏砚若意外退场会重演断代 |

### 场景二完整失败信号

- 延寿丹在所有合理情况下都能无风险完成资源、地图与继任者，成为固定必选项。
- 不使用延寿丹时所有路线必然完全断代，导致保存丹药没有可辩护空间。
- 苏砚的培养只表现为等待6个月，没有稀有资格、资源投入、突破风险和延期后果。
- 引路图或岳徊带队能够跳过硬资格，重新把培养问题压成秘境熟练度问题。
- 引路图与本人带队只有速度数值区别，没有“可跨代但慢”和“依赖活人但快、有产出”的结构差异。
- 苏砚受伤或重伤后，玩家不需要修改任何后续安排。
- 熟练度不足只减少产量，不改变受伤、学习速度与通路状态。
- 第15月所有结果都能被庚金髓总量或一个统一效率分排序，人物、地图、延寿机会与未来断代没有独立意义。

### 场景二下一步验证

先进行设计者内部状态推演，至少覆盖：不延寿并留图、不延寿并保当前取材、延寿且首次突破成功、延寿但普通受伤后第二次成功、延寿但重伤后第二次成功、延寿后候选人身死六条路径。确认时间上都可执行且没有规则矛盾后，再采用“一名独立 GLM-5.2 Pro 只模拟一种玩家偏好”的方式测试选择、情绪、改选节点与后悔，不让单一模型同时分配多类答案。

### 设计者内部六路径检查

以下只检查时间与状态是否合法，不把理想分支当作实际概率结果。

| 路径 | 可执行排程示例 | 第15月主要结果 | 检查结论 |
| --- | --- | --- | --- |
| 不延寿、优先留图、首次突破成功 | 岳徊第1—3月完成地图后寿尽；苏砚第4—6月突破成功，第7—10月研图到熟练度2，第11月起有图独自历练 | 有机会形成“完整接续”，但第3月防御项目延期，且独自历练伤势会继续推迟 | 可执行；保存延寿丹并不等于必然断代，但要接受当前资源损失与后续个人风险 |
| 不延寿、优先当前取材、首次突破成功 | 岳徊第1—3月取得6份庚金髓后寿尽；苏砚第4—6月成功，随后无图独自摸索 | 理想无伤时可在第12月左右达到熟练度3，但通常更慢；最终为“人物接续、知识脆弱”或“有人无路” | 可执行；当前项目按期完成，未来继承高度依赖苏砚本人和连续冒险 |

| 延寿、普通受伤后第二次成功 | 苏砚第6月受伤，第7月休养，第8—10月成功；第11—13月可由岳徊带到熟练度3，只剩第14—15月 | 若地图已在早期完成，可得到完整接续；若早期选择当前取材，则来不及再用3个月完成地图 | 可执行；第一次失败会把开局“取材还是留图”的选择带到结局，不会被一年延寿完全抹平 |
| 延寿、重伤后第二次成功 | 苏砚第6月重伤，第7—8月休养，第9—11月成功；第12—14月可带到熟练度3，第15月只剩一个安排 | 早期有图则完整接续；早期无图则只能稳定当前人物通路，无法同时补齐完整地图 | 可执行；重伤显著压缩交接窗口，但不会自动判定丹药完全浪费 |
| 延寿、候选人身死 | 苏砚第6月身死；岳徊继续活到第15月，可以取材、完成地图或寻找下一名候选人 | 至少可以保留当前资源与路线知识，但没有合格继任者，岳徊寿尽后通路中断 | 可执行；这是延寿丹最大风险，丹药没有失效，却没能换来想要的继承结果 |

内部检查结论：六条路径都能在十五个月内合法执行；不延寿仍存在有图恢复和无图冒险两条可辩护路线，延寿也不会自动保证候选人成功。当前没有发现时间上的严格支配答案。下一步可以进入隔离玩家模拟，但仍需观察“第3月当前防御项目延期”是否足以让早期取材具有真实吸引力。

### 六名独立 GLM-5.2 Pro 玩家模拟

声明：六个 GLM-5.2 Pro 任务相互隔离，每个只知道一种玩家偏好，不知道其他任务与答案。结果用于发现规则问题，不是真人证据。

| 独立玩家偏好 | 前3月主要安排 | 第3月延寿决定 | 核心理由与体验 |
| --- | --- | --- | --- |
| 当前宗门安全 | 连续取材，按期完成当前防御项目 | 使用 | 先保眼前安全，再利用第4—6月候选人突破期补图；希望、责任与押错候选人的内疚并存 |
| 跨代制度建设 | 倾向优先留图 | 使用 | 追求“成熟继任者 + 完整地图”的双重冗余，不愿通路再次依赖单个活人 |
| 对岳徊有感情 | 连续留图，接受当前项目延期 | 使用 | 希望岳徊多活一年、亲眼完成事业与带出继任者，同时担心自己只是把他多出的一年继续当宗门工具 |
| 稀有人才投资 | 连续取材，第4—6月补图 | 使用 | 把丹药押给苏砚的成长窗口，愿意承担第一次失败并给第二次机会，但不因沉没成本无限重试 |
| 延寿丹战略储备 | 连续留图 | 保存 | 先保住不可再生的路线知识，把丹药保留为未来未知关键人物的战略期权；接受当前资源和恢复速度损失 |
| 高风险长期成长 | 连续取材，第4—6月补图 | 使用 | 争取完整接续、成熟继任者、永久地图与长期资源上限；承认候选人身死会使押注落空 |

结果为使用延寿丹5、保存延寿丹1。多数使用并不自动构成失败：延寿丹本来就被确认应拥有本场景最大的收益和机会成本；保存丹药者仍能通过“先留图、候选人以后研图并独自历练”形成明确可辩护路线，因此尚无严格支配。

#### 共同出现的体验信号

- 延寿决定不是单纯购买12个月产量。使用者主要在押“岳徊能否跨过苏砚资格培养期，亲自完成交接”；候选人受伤、重伤或身死会明显改变后悔与后续排程。
- 保存丹药也不是吝啬。该玩家把丹药看成未来可以挽救其他不可替代人物的战略期权，用当前资源延期和苏砚未来的个人风险换取这项选择权。
- 岳徊的延寿同时触发安慰与不适：玩家希望他完成事业，也意识到自己可能把他多出来的一年继续排满取材、带队和留图。
- 苏砚第一次失败后，多数使用丹药的玩家愿意给第二次机会；第二次仍失败时会停止继续押注，说明规则能够制造“继续培养还是接受本轮未成”的新判断，而不只是开局一次选择。
- 资格培养比进入秘境更难的结构被模型正确识别：突破是带死亡与长时间延期的硬门槛，秘境熟练度则是可逐步积累且风险随准备降低的下游阶段。

#### 模拟质量与规则问题

- “跨代制度建设”任务错误地把2个月地图进度当成完整地图；该任务的具体排程无效，只保留其价值取向与延寿/留图选择，整体降权。
- “延寿丹战略储备”任务多次出现文本失控；只采用其重写后清晰的前3月留图、不服丹与四分支排程，情绪细节降权。
- 多个使用延寿丹的任务默认岳徊和熟练度3的苏砚可以在后期分别独立取材，使每月资源翻倍。原规则没有说明沉锋谷每月允许几支队伍进入，这是会显著抬高延寿收益的真实歧义，必须在定稿前封口。
- 只要玩家已经决定服丹，“第1—3月取材、第4—6月趁苏砚突破整理地图”在纯资源与进度上优于先画图。对岳徊有感情的玩家仍选择先留图，说明牺牲当前效率可以表达人物价值；但理性排程中的这一先后关系应向玩家透明。

### 场景二最后一项正式纸面规则：人数、产量与风险

第一轮固定：沉锋谷核心区每月只结算一次宗门历练，秘境当月可取得的庚金髓上限恒定为2份。资源刷新量不随队伍人数增加；岳徊与成熟苏砚同时进入，也不能把同一处资源复制成两条生产线。实际带回0、1或2份，仍由路线熟练度、战斗结算与公开随机结果决定。

队伍人数只通过合格成员提供的战斗能力改变受伤与重伤风险。所有参队者都必须各自满足沉锋谷的灵根、境界、战斗能力与耐力硬资格，不能用高战力引路者把不合格者“带过门槛”。若玩家明知某人不满足硬资格仍强行派入，该人物本次直接重伤濒死、历练中止且不取得资源；这一后果在下令前完全公开，不作为隐藏惩罚。

因此，增加一名合格队员的价值是降低伤势风险、提高任务容错、培养个人熟练度、避免资源通路断代，并让岳徊在不带队的月份可以整理地图或处理其他事务；它不直接提高秘境刷新量。第一版秘境结算表中的概率视为对应队伍组成与熟练度下的公开结果，后续若更换合格队员或战力发生变化，应先公开新的风险分布再让玩家决定。

该规则只固定“地点产出不由人数复制”的核心关系；每月2份、各档概率与具体伤势数值仍是沉锋谷纸面实例参数，不提前升级为所有秘境的统一公式。

### 场景二第一轮裁决

场景二已经回答当前验证问题：寿元会迫使玩家在现任引路者、稀有候选人的培养风险、眼前稳定资源、可跨代路线知识和唯一延寿机会之间作出真实分配。延寿丹收益最大，但没有严格支配保存丹药；继任者的价值来自通路延续与安全冗余，而不是复制资源产量。场景二至此停止继续微调数值，保留真人测试入口，主流程进入三场景联合宗门局面。
